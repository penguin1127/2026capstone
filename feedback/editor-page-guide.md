# 에디터 페이지 UI 피드백 가이드

## 색상 조정

- 캔버스 색상: 흰색 → 약간 회색으로 변경
- 캔버스 뒷배경: 진한 회색 → 연한 회색으로 변경

---

## 사이즈 조정

- 좌측 메뉴들: 현재 대비 1.5~2배 키우기
- 좌하단 상태바: 현재 대비 1.5배 키우기
- 전체적인 메뉴 (우상단 버튼 3개 제외): 현재 대비 1.5배 키우기

---

## 레이아웃 조정

- 애니메이션 창 위치: 우하단 → 우측 메뉴바 좌측으로 이동
