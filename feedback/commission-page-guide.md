# 커미션 상세 페이지 UI 가이드

## 핵심 방향

이 플랫폼의 커미션 페이지는 **마켓플레이스형 상품 전시** 방식으로 구성한다.
Patreon처럼 "플랜 선택" 형태의 SaaS 구독 UI는 최대한 지양한다.
Skeb, VGen, DeviantArt처럼 작가가 자신의 커미션 서비스를 상품처럼 등록하고,
의뢰자가 보고 선택하는 방식이다.

---

## 레이아웃 구조

**좌측 메인 영역 (콘텐츠)**
- 작가 프로필 (썸네일, 닉네임, 팔로우 버튼)
- 커미션 제목 및 설명
- 샘플 이미지들
- 작업 범위 및 조건
- 유의사항

**우측 사이드바 (액션)**
- 가격
- 수락 상태 (오픈 / 마감)
- 예상 납기일
- "의뢰하기" 버튼

---

## 피해야 할 것

- "플랜 선택", "Basic / Pro / Premium" 같은 구독 서비스 느낌의 표현
- 가격 비교표 형태의 플랜 카드 나열 (Patreon, 구독 SaaS 스타일)
- 페이지 중앙에 플랜을 나란히 배치하는 레이아웃

---

## 참고 사이트

- **Skeb** — 심플한 작가 페이지 + 우측 가격/요청 버튼
- **VGen** — 서비스 카드 목록 + 상세 페이지 우측 사이드바
- **DeviantArt** — 작가 프로필 내 커미션 위젯
